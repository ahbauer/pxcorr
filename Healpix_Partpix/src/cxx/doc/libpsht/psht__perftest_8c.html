Detailed Description

Performance test of libpsht's map synthesis and analysis algorithms.

This program creates a list of transform jobs with a user-specified maximum multipole lmax that operate on a HEALPix, ECP or Gaussian grid. Depending on the geometry type, the user has to specify the Nside parameter (for HEALPix) or the number of pixels per ring. The individual job types are also given by the user and can be "alm2map", "map2alm", "alm2map_pol", "map2alm_pol", "alm2map_spin[1-3]", "map2alm_spin[1-3]", and "alm2map_deriv1". Any combination of job types is allowed.

All requested jobs are executed simultaneously.
