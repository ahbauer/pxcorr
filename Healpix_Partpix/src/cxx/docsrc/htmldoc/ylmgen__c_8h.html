void Ylmgen_init ( Ylmgen_C *  gen,
int  l_max,
int  m_max,
int  spinrec,
double  epsilon 
)

Creates a generator which will calculate Y_lm(theta,phi=0) up to l=l_max and m=m_max. It may regard Y_lm whose absolute magnitude is smaller than epsilon as zero. If spinrec is nonzero, the spin-1 and spin-2 Y_lm will be calculated by recursion from the spin-0 ones, otherwise Wigner d matrix elements will be used.
